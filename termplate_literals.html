TEMPLATE LITERALS

Template literals use backticks ` instead of single or double quotes

ES6 introduced template literals that :-

  • made easy writing multiline strings and keep all white space when printed
  • or strings with variables. Variables are put directly inside ${} withour concatenations
  • When you want to use certain escape sequences such as new lines or tabs

Tagged Template Literals

Tagged template literals gives us the ability to runa function on the content of template literals. For example, String.raw`hs\njfh\tsjf` which skips the execution of escape sequence and turn the whole literal to anormal string

Uses of tagged template literals

  • Bolding all values with tagged template literals
  • Removing newlines from multiline strings
  • querying elements with tagged template literals
